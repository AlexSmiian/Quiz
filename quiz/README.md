# Архітектура

- Vite — надшвидкий Dev Server

- React + SWC — рендеринг UI та оптимізована збірка

- TypeScript

- SCSS Modules — інкапсульовані стилі

- React Router — навігація між екранами

- Custom Hooks — логіка питань, прогрес, відповіді

- State — useState/useReducer

- Анімації кнопок та переходів між питаннями (CSS/SASS анімації)

# Реалізовано двома способами

### 1. Гілка master — керування станом через кастомний хук

### 2. Гілка context — глобальний стан через React Context
